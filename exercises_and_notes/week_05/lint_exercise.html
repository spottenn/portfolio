Lint Exercise

  • Researched linting packages and decided on ESLint due to its customization options and integration with my IDE of choice for web development JetBrains PhpStorm.
  • Installed Node.js, and therefore npm, on my local machine.
  • Ran npm init on portfolio project
  • Installed ESLint as a local development dependency using npm.
  • Ran eslint --init on portfolio project
  • Enabled ESLint in PHPStorm using default configuration, which also means that all new JavaScript files will be analysed by ESLint in the editor as they are typed.
  • Used ESLint and PhpStorm code inspection to check and fix the existing JavaScript in this portfolio.
  • Added the proper files to version control (Git) and the portfolio repository to enable easy installation/initialization in the future
